Helper to find locally installed copies of [CoffeeScript][coffeescript].

## Install
```bash
$ npm install find-coffee --save
```

## Usage
```javascript
import findCoffee from 'find-coffee'

// Find CoffeeScript v2
const coffee = findCoffee(2)
```
